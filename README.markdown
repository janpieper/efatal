efatal Log Viewer
=================
To use this small quick & dirty app you need documents like this one:

    {
        "type"    : "error",
        "date"    : "2011-10-18T00:57:30+01:00"
        "error"   : "E_NOTICE",
        "file"    : "/var/www/index.php",
        "line"    : 42,
        "message" : "Undefined variable: foobar"
    }
